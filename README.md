# Sync Skill to Claude Desktop

A Claude Code skill for uploading Claude Code skills to claude.ai, making them available in Claude Desktop via cloud sync.

## Documentation

See [SKILL.md](./SKILL.md) for complete documentation and usage instructions.

## Installation

```bash
npx skills add HartreeWorks/skill--sync-skill-to-claude-desktop
```

Or install all HartreeWorks skills: `npx skills add HartreeWorks/skills`
